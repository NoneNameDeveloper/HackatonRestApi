Для запуска требуется python 3.10+
pip install -r requirements.txt

Создать config.env (по образцу из config.env.example) и наполнить данными
Запуск апи:
```bash
python main.py
```

Запуск тестового бота (предлагается поменять токены):
```bash
cd example-bot
sh start-bot.sh
```
